# DataVisualization
Big data analysis and advanced data visualization pipeline for large-scale datasets to answer socioeconomic research questions.


## Data Sources & Licensing

The underlying datasets used in these projects are sourced from publicly available repositories, including:
* **United States Census Bureau** (US Census Bureau) - Status of Health Insurance, Homeless Rate, Available Housing, 
* **Wonder CDC** (Wonder CDC) - Underlying Causes of Death
* Various other open-source public data portals.

All datasets remain the property of their respective creators and are used here strictly for analytical and educational purposes.
